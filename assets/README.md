# 素材目录说明

桌面宠物会自动扫描这两个文件夹，启动后右键宠物 →「重新加载素材」即可让新放入的文件生效。

## 图片 → `images/`

- 把你的图片放进 `images/` 文件夹。
- 支持格式：`png` / `jpg` / `jpeg` / `gif` / `webp` / `svg`。
- 点击宠物时会随机切换到其中一张。
- 自带一套「小黑」风格形象：`xiaohei_idle / xiaohei_sing / xiaohei_mic / xiaohei_bounce / xiaohei_note`（黑色实心、白点眼、细腿、空表情，透明背景便于浮在桌面）。点击会在这几个动作间循环切换。
- 原来的彩色圆脸形象已挪到 `assets/images_backup/`（含 `_placeholder.svg`）。想换回去把文件挪回 `images/` 即可。

## 音频 → `audio/`

- 把音频片段放进 `audio/` 文件夹。
- 支持格式：`mp3` / `wav` / `ogg` / `m4a` / `aac`。
- 点击宠物时会随机播放其中一段。

### 命名规范（重要）

**音频文件名（去掉扩展名）会作为宠物头顶气泡显示的“台词”。**

例如，把一段音频命名为：

```
我们的爱.mp3
```

点击宠物播放这段音频时，头顶气泡就会显示「我们的爱」。所以建议用你想显示的那句话来给音频文件命名。
